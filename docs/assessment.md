# Assessment and Evidence

Evidence is collected at the end of each day. Keep it in a local `evidence/` folder or a shared drive.

Daily checklist
Day 1
- inventory.csv and inventory.md
- connectivity report

Day 2
- snmp_summary.json
- syslog_parsed.json
- obs_traffic.pcap
- obs_traffic_summary.txt

Day 3
- api_normalized.json
- report.md

Day 4
- ansible check mode diff
- apply summary

Day 5
- plan artifacts
- apply artifacts
- recap.md
