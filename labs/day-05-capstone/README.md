# Day 5 - Intent Capstone

Outcomes
- Express intent as data.
- Generate a plan, review evidence, and apply changes.

Capstone flow
1. Go to `modules/netintent-ops`.
2. Follow the quick start in `modules/netintent-ops/README.md`.
3. Submit a plan using `docs/sample_plan_request.json`.
4. Review artifacts in `modules/netintent-ops/artifacts/<plan_id>/`.

Stretch goals
- Add validation checks to the plan playbook.
- Add one new intent field and render it in templates.

Evidence
- Plan request and plan response.
- Artifacts folder with rendered configs and evidence.
